Modules

Extension des fonctionnalités de l'API de scripting de nanos world

Les modules vous permettent d'étendre les fonctionnalités de l'API de scripting de nanos world côté serveur. Actuellement, seuls les modules C sont pris en charge.

caution

Sachez que cette partie de l'API de scripting est actuellement en cours de réalisation et peut être modifiée à tout moment.

info

Un guide pour créer votre propre module suivra.

Liste des modules pris en charge​

LuaSQL-MySQL​

LuaSQL-MySQL est une interface simple pour communiquer avec un serveur MySQL ou MariaDB. Nous fournissons une version précompilée de ce module qui est déjà prête à être utilisée.

Documentation de LuaSQL : http://keplerproject.github.io/luasql/

Téléchargement & Installation​

Téléchargez la bibliothèque de votre plateforme :

  • mysql_windows.zip
  • mysql_linux.zip

Extractez-le dans votre dossier ServerRoot/Modules/ :

Exemple d'utilisation​

Server/Index.lua
local luasql = require("luasql.mysql")
local mysql = luasql.mysql()
local connection = mysql:connect("database", "username", "password", "localhost", 3306)

if not connection or not connection:ping() then
    Package.Log("Connexion échouée")
else
    connection:execute([[
        CREATE TABLE hello_world(
            message varchar(50)
        );
    ]])

    connection:execute(string.format([[
        INSERT INTO hello_world VALUES ('%s');
    ]], connection:escape("Hello nanos.world")))
    -- (Vous devez toujours utiliser escape pour la saisie de l'utilisateur afin d'éviter l'injection sql !)

    local cursor = connection:execute("SELECT * FROM hello_world")
    local row = cursor:fetch({}, "a")

    Package.Log("Message: " .. tostring(row.message))
    -- Outputs: "[Script]   Message: Hello nanos.world" to the console
end
info

Astuce: Toujours échapper tous les paramètres de la requête avec la fonction :escape!
